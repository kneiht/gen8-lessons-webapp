Part 1: Key Notes (Tóm tắt kiến thức)

1. Like (giống như)

Used to show similarity between two things.

Example: Yoga is like meditation for the body and mind.

Structure: Subject + be + like + noun/pronoun

2. Different from (khác với)

Used to show difference between two things.

Example: Horse riding is different from jogging because it requires more equipment.

Structure: Subject + be + different from + noun/pronoun

3. (not) as...as (bằng/không bằng)

Used to compare equality or inequality between two things.

Example: Making models is not as popular as gardening among older people.

Structure: Subject + be + (not) as + adjective + as + noun/pronoun

4. Important Notes

  • Use adjectives between "as" and "as"
  • "Different from" is more common than "different than"
  • "Like" is a preposition, not a conjunction

Exercise 1: Fill in the blanks with "like", "different from", or "as...as"

👉 Complete the sentences using the correct comparative structure:

1. Making models is ___________ (not/popular) gardening among adults.
2. A dollhouse is ___________ a real house but much smaller.
3. Horse riding is completely ___________ jogging - you need special equipment.
4. This insect is ___________ (not/big) that bug over there.
5. Yoga can be ___________ meditation in terms of reducing stress.
