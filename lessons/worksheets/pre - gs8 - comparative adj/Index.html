Phần 1: Tóm tắt kiến thức (Key Notes)

1. Khái niệm

Tính từ so sánh hơn dùng để so sánh hai người, vật, hoặc sự việc, thường có nghĩa là "hơn".

2. Công thức chung

🔹 Tính từ ngắn (1 âm tiết, hoặc 2 âm tiết kết thúc bằng -y):

👉 Tính từ + -er + than

  • tall → taller than (cao hơn)
  • small → smaller than (nhỏ hơn)
  • happy → happier than (vui hơn)

3. Quy tắc đặc biệt

  • Nếu tận cùng bằng "e" → thêm "-r": large → larger than
  • Nếu tận cùng là 1 nguyên âm + 1 phụ âm → gấp đôi phụ âm + "-er": big → bigger than
  • Tính từ tận cùng "-y" → đổi "y" thành "i" + "-er": easy → easier than

4. Tính từ dài (2 âm tiết trở lên)

👉 more + tính từ + than

  • beautiful → more beautiful than (đẹp hơn)
  • expensive → more expensive than (đắt hơn)
  • interesting → more interesting than (thú vị hơn)

5. Tính từ bất quy tắc

Tính từ Dạng so sánh hơn
good/well better
many/much more
little less
far farther/further
narrow narrower
clever cleverer
