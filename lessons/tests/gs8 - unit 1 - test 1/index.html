Exercise 3: Choose the correct answer

1. They're really ___ organizing parties for their friends.

Correct answer: c) into

2. Tom is crazy ___ watching comedy movies every weekend.

Correct answer: d) about

3. I dislike ___ emails because I prefer phone calls.

Correct answer: a) sending

4. Do you fancy ___ badminton this weekend?

Correct answer: d) playing

5. I'm fond ___ improving my physical health through sport.

Correct answer: b) of

6. We prefer ___ with our neighbours to staying isolated.

Correct answer: b) connecting

7. She loves ___ a beautiful dress for the costume party.

Correct answer: a) wearing / to wear

8. I enjoy ___ my computer skills to help others.

Correct answer: c) using

9. My brother detests ___ cruel to animals and always helps them.

Correct answer: a) being

10. She's keen ___ building a DIY kit for her projects.

Correct answer: a) on

Exercise 4: Rearrange the words to make meaningful sentences

1. DIY / in / fond / my / doing / I'm / of / projects / spare / time.

Correct answer: I'm fond of doing DIY projects in my spare time.

2. for / enjoys / her / knitting / she / family / scarves.

Correct answer: She enjoys knitting scarves for her family.

3. every / friends / crazy / about / is / Tom / weekend / with / his / playing / volleyball.

Correct answer: Tom is crazy about playing volleyball with his friends every weekend.

4. when / detest / | / ingredients / preparing / cooking.

Correct answer: I detest cooking when preparing ingredients.

5. get / keen / my / is / on / sister / to / some / visiting / the / new / park / fresh / air.

Correct answer: My sister is keen on visiting the new park to get some fresh air.

6. do / going / fancy / you / this / Sunday / skateboarding / ?

Correct answer: Do you fancy going skateboarding this Sunday?
